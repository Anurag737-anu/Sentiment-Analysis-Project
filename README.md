## Twitter Sentiment Analysis

This project performs sentiment analysis on tweets using NLP and Machine Learning.

## Features
- Text Cleaning
- WordCloud Visualization
- Sentiment Classification
- Logistic Regression
- Confusion Matrix
- Accuracy Evaluation

- Pandas
- NumPy
- Scikit-learn
- NLP
- Matplotlib
- Seaborn
- Google Colab
